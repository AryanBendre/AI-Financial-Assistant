# 💰 FinsightAI: Personal Financial Awareness Assistant

FinsightAI is a comprehensive full-stack web application designed to help users track monthly finances and gain AI-driven insights into their spending habits. By utilizing a hybrid rule-based and AI categorization engine, it empowers users to achieve better financial literacy through the **50:30:20 budgeting rule**.

View the deployed project on **Hugging Face Spaces**.

## 🛠️ Technical Stack
- **Backend:** Python (FastAPI) for high-performance, asynchronous API handling  
- **Database:** PostgreSQL hosted on Supabase for persistent, "always-on" storage  
- **AI Engine:** Google Gemini 2.5 Flash for intelligent expense classification and advisory  
- **Frontend:** Vanilla HTML5, CSS3 (Glassmorphism), and JavaScript  
- **Deployment:** Dockerized and hosted on Hugging Face Spaces  
- **ORM:** SQLAlchemy for database modeling and management  

---

## 🌟 Core Features
- **User Management:** Unique profile creation based on name and age  
- **Income Tracking:** Seamless logging of monthly in-hand income  
- **AI Categorization:** Automatic classification of expenses into Needs, Wants, or Investments using a multi-layered logic  
- **Financial Analysis:** Real-time 50:30:20 budget breakdown based on actual spending  
- **AI Advisory:** Personalized, non-judgmental financial tips generated by Gemini based on your specific data  
- **Full CRUD:** Add, view, update, and delete individual expense records with instant UI updates
